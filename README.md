# Women Matter | Igniting Change

**Women Matter** is a modern, responsive frontend web application developed as a semester project. The platform is dedicated to celebrating and highlighting the remarkable achievements of women in Pakistan, providing a voice across sectors like Education, Technology, Sports, and Leadership.

## Project Overview

This project is built to demonstrate proficiency in modern frontend development techniques, focusing on user experience (UX), responsive design, and clean code architecture.

### Key Features
* **Dynamic Hero Slider:** A high-impact hero section featuring a JavaScript-powered image slider with smooth transitions.
* **Sticky Sidebar:** A functional "Aside" in the blog section that remains fixed while scrolling, enhancing navigation for long-form content.
* **Responsive Layout:** Fully optimized for all screen sizes (Desktop, Tablet, and Mobile) using CSS Flexbox and Grid.
* **Interactive Elements:**
   * Custom hover effects on all call-to-action (CTA) buttons.
   * Multi-level dropdown navigation menu for blog categories.
* **Clean UI/UX:** A professional color palette and typography using Google Fonts (Montserrat).

---

## Technologies Used

* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Advanced layout techniques (Grid & Flexbox) and Media Queries.
* **JavaScript:** Used for UI interactivity and the hero slider logic.
